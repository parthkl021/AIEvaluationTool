# AI Evaluation Tool UI

This section documents the web interface used to configure, start, monitor, and inspect evaluation runs.

The UI in this repository is the Test Case Execution Dashboard. It provides a guided way to launch new runs, track execution in real time, and review evaluation outcomes without relying only on CLI commands or database inspection.

## What The Dashboard Provides

- run configuration through a web interface
- real-time tracking of evaluation progress
- detailed run-level and testcase-level inspection
- structured result review with better visibility into execution state

## Typical Usage Flow

- Start the frontend and backend services.
- Open the dashboard in the browser.
- Create or continue a run.
- Monitor progress as the execution moves forward.
- Review run details and testcase-level results.
