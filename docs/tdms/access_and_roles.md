# TDMS Access And Roles

After starting TDMS, open the frontend in your browser and sign in with the role-based accounts configured for the system.

## Default Roles

| Role | Username | Password |
|------|----------|----------|
| Admin | `admin` | `admin123` |
| Manager | `manager` | `manager123` |
| Curator | `curator` | `curator123` |
| Viewer | `viewer` | `viewer123` |

## Typical Usage

- administrators manage the full system
- managers coordinate operational activity
- curators maintain test data quality
- viewers access read-only information
