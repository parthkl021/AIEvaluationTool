# TDMS

This section documents the Test Data Management System.

## What TDMS Provides

TDMS is the web-based application used to manage the structured data required for evaluation workflows, including test cases, prompts, responses, and supporting metadata.

## Architecture Summary

- Frontend: React, TypeScript, Vite
- Backend: FastAPI
- Database: SQLite or MariaDB
